2. Текст помилки типізації (Частина 4)

Типова помилка типізації, що виникає при спробі передати number (наприклад, 1) до пропсу, який очікує boolean:

// [vue-tsc] error TS2322: Type 'number' is not assignable to type 'boolean'.


Коментар, що ілюструє цю помилку, додається до файлу ContactView.vue:
<!-- <UserCard username="Student_1" :isActive="1" /> // Type 'number' is not assignable to type 'boolean'. -->


3. Висновки: що дає використання TypeScript та Vue Router у проєкті?

Використання цих інструментів забезпечує професійний рівень розробки та підвищує якість кінцевого продукту:

TypeScript (TS)

1. Дозволяє виявляти помилки невідповідності типів (як, наприклад, передача числа замість boolean) на етапі компіляції, запобігаючи помилкам під час виконання.

2. Забезпечує статичний аналіз коду, покращує автодоповнення в редакторі (IDE) та спрощує подальший рефакторинг проєкту.

Vue Router

1. Створює односторінковий застосунок, що забезпечує швидкі, безперервні переходи між сторінками, значно покращуючи користувацький досвід (UX).

2. Надає стандартизований та чіткий механізм для керування навігацією, мапуючи URL-адреси на відповідні компоненти сторінок.
